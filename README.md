# 🚀 Habit Tracker

A modern and responsive Habit Tracker built with **Next.js** and **Tailwind CSS** to help users build consistency by tracking daily habits. The application provides a clean interface for managing habits and monitoring daily progress.

---

## ✨ Features

- ✅ Add and manage daily habits
- 📊 Track your daily progress
- 🎯 Build consistency with habit tracking
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- ⚡ Fast performance with Next.js

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| Next.js | Frontend Framework |

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/mohityadav24952-ai/Habit-Tracker.git
```

### 2️⃣ Navigate to the project folder

```bash
cd Habit-Tracker
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm run dev
```

### 5️⃣ Open your browser

Visit:

```
http://localhost:3000
```
